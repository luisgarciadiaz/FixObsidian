# FixObsidian

Repair and improve Obsidian markdown notes generated by the Vortexy File Sorter.

## Quick start

```powershell
pip install python-dotenv  # optional, for .env support
python tools/fix_obsidian_notes.py --dry-run --limit 100
python tools/fix_obsidian_notes.py --limit 500
```

## Configuration

1. Edit `config.json` — set your vault and library paths, subfolder map, etc.
2. (Optional) Copy `.env.example` to `.env` and set your paths for convenience.
3. CLI flags override both.

## Features

- Fixes filenames to `Author - Title.md` format
- Adds `title` and `tags` to frontmatter (was missing)
- Adds clickable Open Library ISBN links
- Moves notes into genre subfolders by category (`--organize`)

See `agents.md` for full documentation.
